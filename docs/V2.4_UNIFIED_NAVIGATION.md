# ShieldNet v2.4 — Unified Navigation & Platform Dashboard

## Changes

- unified platform and guild navigation;
- context-aware current-guild menu;
- active route highlighting;
- direct links to Members, Moderation, Verification, Permissions, Audit and Server Control;
- fixed duplicate Angular route for `/guild/:guildId`;
- Server Control now has the explicit route `/guild/:guildId/control`;
- responsive horizontal navigation for mobile screens;
- platform links are shown only to SuperAdmin users.

No database migration is required for this release.
