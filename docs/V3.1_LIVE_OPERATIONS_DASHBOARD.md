# ShieldNet v3.1 Live Operations Dashboard

Adds a SuperAdmin/Platform Admin live control plane at `/platform/operations`.

Features:
- authenticated WebSocket telemetry every 5 seconds;
- backend and PostgreSQL status with latency;
- Valkey status, latency, memory and Discord queue depth;
- backend/bot/scheduler heartbeat state;
- recent audit event stream;
- automatic WebSocket reconnect;
- HTTP snapshot fallback.

Nginx must allow WebSocket upgrade for `/api/` proxy traffic.
