# ShieldNet v1.7 — Member Watchlist & Risk Review

## Added

- Watchlist flag for individual Discord members.
- Risk levels: low, medium, high and critical.
- Review reason and next review date.
- Dedicated `Watchlist` and `Review due` filters.
- Dashboard counters for watchlisted members, high-risk members and overdue reviews.
- Risk indicators directly in the member list.
- PostgreSQL migration `0019_member_watchlist_risk_review`.

## Upgrade

```bash
cd /opt/shieldnet/backend
source venv/bin/activate
alembic upgrade head
sudo systemctl restart shieldnet-backend

cd /opt/shieldnet/admin-frontend
npm ci
npm run build
```

Publish `admin-frontend/dist/shieldnet-admin/browser/` using the existing Nginx deployment process.
