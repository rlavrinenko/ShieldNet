# ShieldNet v4.3 Permission Simulator

The simulator resolves a member's effective Discord permissions for a selected channel using the Discord precedence order:

1. `@everyone` guild permissions
2. merged member role permissions
3. `@everyone` channel overwrite
4. aggregated role overwrites (deny, then allow)
5. direct member overwrite
6. owner or Administrator bypass

Discord Worker synchronizes channel overwrites as part of the existing Explorer snapshot. The page is available at `/guild/{guild_id}/permission-simulator`.

After installation, restart `shieldnet-bot` or wait for the next Explorer synchronization before trusting channel-specific results.
