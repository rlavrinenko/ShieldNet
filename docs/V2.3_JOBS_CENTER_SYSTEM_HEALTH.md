# ShieldNet v2.3 — Jobs Center & System Health

Adds a SuperAdmin-only operational console at `/platform/jobs`.

## Included safe jobs

- Database probe
- Guild registry snapshot
- Member index snapshot
- Audit snapshot

Each manual execution is persisted in `system.job_runs` and written to the audit log. Jobs are intentionally read-only diagnostics and execute inline; they do not run shell commands or destructive cleanup.
