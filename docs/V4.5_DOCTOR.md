# ShieldNet v4.5 Doctor & Diagnostics Center

Web page: `/platform/doctor`

Root CLI:

```bash
sudo /opt/shieldnet/scripts/shieldnet-doctor.sh /opt/shieldnet
```

The web report validates application-visible configuration, database schemas and table privileges, Alembic, runtime heartbeats and Valkey. The root CLI additionally validates systemd, filesystem traversal, Nginx, WebSocket headers and local service executables.
