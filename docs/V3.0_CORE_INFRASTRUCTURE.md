# ShieldNet v3.0 Core Infrastructure

This release turns the existing Discord code into installable runtime services.

Services:
- shieldnet-backend.service
- shieldnet-bot.service
- shieldnet-scheduler.service
- redis.service or redis-server.service

The installer creates isolated Python virtual environments, preserves existing environment files, installs dependencies, applies Alembic migrations, and validates every service before enabling it.

Redis queue: `shieldnet:discord:jobs`

Supported worker jobs:
- sync_guilds
- sync_roles
- sync_members
- security_scan

Runtime heartbeat API:
- POST /api/v1/internal/runtime/heartbeat
- GET /api/v1/platform/runtime
